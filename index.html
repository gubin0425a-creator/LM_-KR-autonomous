대화창 중심으로 유지하면서, 앱 단위 저장을 붙였습니다.

모바일은 무료 5개 앱, PC는 최대 1000개 앱까지 만들고 다시 불러올 수 있습니다. 마지막 채팅 날짜로도 검색되고, 대화 내용 일부로도 검색됩니다. 72시간 지난 대화는 자동 최적화되고, 다시 들어오면 복원해서 이어서 사용할 수 있습니다.

기기 확인 중…
기본 모델: openrouter/auto
채팅 전용 UI
OpenRouter API 키를 저장하면 바로 대화할 수 있습니다.

설정

키를 한 번 저장하면 업데이트 후에도 유지됩니다. 입력칸을 비워두고 저장하면 기존 키를 그대로 유지합니다.

여기에는 기본 모델 목록 + 직접 추가한 최대 8개 모델이 같이 보입니다. 아무것도 안 고르면 오토라우터가 기본입니다.

비워두면 일반 채팅이 아니라, 앱/게임/웹사이트 제작을 바로 도와주는 기본 빌더 프롬프트가 적용됩니다.

로컬에 저장된 모든 대화와 설정 데이터를 하나의 JSON 백업 파일로 내보내거나 가져옵니다.
